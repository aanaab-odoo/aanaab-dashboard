// ============================================= // doGet — Web App API v3 // الإضافة دي بس — حطها في نهاية الـ Apps Script // لا تغير أي حاجة في الكود القديم // ============================================= function doGet(e) { var type = e && e.parameter && e.parameter.data ? e.parameter.data : 'all'; var result = {}; try { if (type === 'pl' || type === 'all') result.pl = getPLData(); if (type === 'rsa' || type === 'all') result.rsa = getRSAData(); if (type === 'bs' || type === 'all') result.bs = getBSData(); if (type === 'tb' || type === 'all') result.tb = getTBData(); } catch(err) { result.error = err.toString(); } return ContentService .createTextOutput(JSON.stringify(result)) .setMimeType(ContentService.MimeType.JSON); } function getPLData() { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName("P&L"); if (!sheet) return {error: "P&L sheet not found"}; var lastRow = sheet.getLastRow(); var lastCol = sheet.getLastColumn(); // data[0] = header, data[1] = Revenue, data[2] = COGS, etc. var data = sheet.getRange(1, 1, lastRow, lastCol).getValues(); // Dates: row 0, starting col index 2 var headers = data[0]; var dates = []; for (var i = 2; i < headers.length; i++) { var h = headers[i]; var label = h instanceof Date ? Utilities.formatDate(h, Session.getScriptTimeZone(), "MMM yyyy") : String(h || '').trim(); if (label && label !== 'null') dates.push({label: label, i: i}); } function rowVals(rowIdx) { var row = data[rowIdx]; if (!row) return dates.map(function() { return 0; }); return dates.map(function(d) { return parseFloat(String(row[d.i] || 0).replace(/,/g,'')) || 0; }); } // Net Profit = last row in the sheet (آخر صف فيه بيانات) // data.length - 1 = آخر index var npIdx = data.length - 1; // تأكد إن فيه بيانات while (npIdx > 1) { var hasData = dates.some(function(d) { return Math.abs(parseFloat(String(data[npIdx][d.i] || 0).replace(/,/g,'')) || 0) > 0.01; }); if (hasData) break; npIdx--; } return { dates: dates.map(function(d) { return d.label; }), revenue: rowVals(1), // row 2 في الشيت cogs: rowVals(2), // row 3 grossProfit: rowVals(3), // row 4 salaries: rowVals(4), // row 5 marketing: rowVals(5), // row 6 otherOpex: rowVals(8), // row 9 ebitda: rowVals(10), // row 11 otherIncome: rowVals(13), // row 14 netProfit: rowVals(npIdx), // آخر row فيه بيانات _debug: {totalRows: data.length, npRowUsed: npIdx + 1} // للتشخيص }; } function getRSAData() { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName("Revenue stream analysis"); if (!sheet) return {error: "RSA sheet not found"}; var lastRow = sheet.getLastRow(); var lastCol = sheet.getLastColumn(); var data = sheet.getRange(1, 1, lastRow, lastCol).getValues(); // Dates في row 0 من col 1 var headers = data[0]; var dates = []; for (var i = 1; i < headers.length; i++) { var h = headers[i]; var label = h instanceof Date ? Utilities.formatDate(h, Session.getScriptTimeZone(), "MMM yyyy") : String(h || '').trim(); if (label && label !== 'null') dates.push({label: label, i: i}); } function rowVals(rowIdx) { var row = data[rowIdx]; if (!row) return dates.map(function() { return 0; }); return dates.map(function(d) { return parseFloat(String(row[d.i] || 0).replace(/,/g,'')) || 0; }); } function rowName(rowIdx) { if (!data[rowIdx]) return ''; return String(data[rowIdx][0] || '').replace(/^"|"$/g,'').trim(); } // B2C: rows 1-5 var b2c = []; for (var i = 1; i <= 5; i++) { var n = rowName(i); if (n && n !== 'null') b2c.push({name: n, vals: rowVals(i)}); } // B2B: rows 9-11 var b2b = []; for (var i = 9; i <= 11; i++) { var n = rowName(i); if (n && n !== 'null') b2b.push({name: n, vals: rowVals(i)}); } // B2G projects + total // ابحث عن row اسمه "B2G" = هو الـ total var b2gProjects = []; var b2gTotalVals = null; for (var i = 15; i < Math.min(data.length, 60); i++) { var n = rowName(i); if (!n || n === 'null' || n === '') continue; var vals = rowVals(i); var hasData = vals.some(function(v) { return Math.abs(v) > 0.01; }); if (!hasData) continue; if (n === 'B2G') { b2gTotalVals = vals; // هذا هو الـ total } else { b2gProjects.push({name: n, vals: vals}); } } // لو مفيش "B2G" total row، احسبه من الـ projects if (!b2gTotalVals) { b2gTotalVals = dates.map(function(_, di) { return b2gProjects.reduce(function(s, p) { return s + (p.vals[di] || 0); }, 0); }); } return { dates: dates.map(function(d) { return d.label; }), b2c: b2c, b2b: b2b, b2g: b2gProjects, b2gTotal: b2gTotalVals }; } function getBSData() { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName("Balance Sheet"); if (!sheet) return {error: "Balance Sheet not found"}; var lastRow = sheet.getLastRow(); var lastCol = sheet.getLastColumn(); var data = sheet.getRange(1, 1, lastRow, lastCol).getValues(); var headers = data[0]; var dates = []; for (var i = 4; i < headers.length; i++) { var h = headers[i]; var label = h instanceof Date ? Utilities.formatDate(h, Session.getScriptTimeZone(), "MMM yyyy") : String(h || '').trim(); if (label && label !== 'null') dates.push({label: label, i: i}); } var accounts = []; for (var r = 1; r < data.length; r++) { var row = data[r]; if (!row[0]) continue; var type = String(row[3] || '').trim(); if (!type) continue; var vals = {}; dates.forEach(function(d) { vals[d.label] = parseFloat(String(row[d.i] || 0).replace(/,/g,'')) || 0; }); accounts.push({id: row[0], code: parseInt(row[1])||0, name: String(row[2]||'').trim(), type: type, vals: vals}); } return {dates: dates.map(function(d){return d.label;}), accounts: accounts}; } function getTBData() { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName("Trial Balance"); if (!sheet) return {error: "Trial Balance not found"}; var lastRow = sheet.getLastRow(); var lastCol = sheet.getLastColumn(); var data = sheet.getRange(1, 1, lastRow, lastCol).getValues(); var headers = data[0]; var dates = []; for (var i = 4; i < headers.length; i++) { var h = headers[i]; var label = h instanceof Date ? Utilities.formatDate(h, Session.getScriptTimeZone(), "MMM yyyy") : String(h || '').trim(); if (label && label !== 'null') dates.push({label: label, i: i}); } var accounts = []; for (var r = 1; r < data.length; r++) { var row = data[r]; var code = parseInt(row[1]) || 0; if (!code) continue; var vals = {}; dates.forEach(function(d) { vals[d.label] = parseFloat(String(row[d.i] || 0).replace(/,/g,'')) || 0; }); accounts.push({id: row[0], code: code, vals: vals}); } return {dates: dates.map(function(d){return d.label;}), accounts: accounts}; }
